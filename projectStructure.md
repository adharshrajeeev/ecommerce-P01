Build a modern eCommerce platform MVP using:

* Next.js 15 App Router
* TypeScript
* Tailwind CSS
* Shadcn/UI
* Supabase (Auth + Database + Storage)
* React Query
* Zustand

The application must be fully responsive for:

* Mobile
* Tablet
* Desktop

Use a reusable plug-and-play architecture with feature modules and service layers.

Project Structure:

src/
├── app/
├── components/
├── features/
├── services/
├── hooks/
├── store/
├── lib/
├── types/
└── utils/

Use clean code practices and reusable components.
